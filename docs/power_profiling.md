# Power Profiling (External Project)

Detailed power monitoring and analysis have been moved out of this repository.

For CPU/GPU/system power monitoring on REPACSS, TimescaleDB-based queries, rack-level power
validation, and Excel report generation, please use the dedicated project
[`Repacss-power-profiling`](https://github.com/billzyj/Repacss-power-profiling).

That project provides:

- A Python client and CLI for querying iDRAC and infrastructure power metrics
- Node-level and rack-level power/energy analysis
- Configurable SSH tunneling and database access
- Excel and CSV reporting

This repository is now focused on **benchmarks and experiment data only**.
